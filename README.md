# 📁 Storage Management Server

A scalable backend system for managing user file storage with upload, delete, and storage tracking functionality.

Built using **Node.js, Express, TypeScript, Prisma, PostgreSQL, Cloudinary, and Multer**.

---

## 🚀 Features

- 🔐 User Authentication (JWT based)
- 📤 Upload Image / PDF (Cloudinary)
- 🗑 Delete File (Auto remove from Cloudinary)
- 📊 User Storage Limit (15GB per user)
- 📉 Auto Storage Tracking (increment/decrement)
- 📅 Date-wise File Filtering
- 📦 MVC Architecture (Controller, Service, Route)
- 🛡 Secure API (Protected routes)

---

## 🏗 Tech Stack

- **Backend:** Node.js, Express.js
- **Language:** TypeScript
- **Database:** PostgreSQL
- **ORM:** Prisma
- **File Storage:** Cloudinary
- **Upload Middleware:** Multer
